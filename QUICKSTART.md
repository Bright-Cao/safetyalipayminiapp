## 📋 3步快速配置

### 步骤1：开通云开发（1分钟）

1. 打开微信开发者工具
2. 点击顶部工具栏的「**云开发**」按钮
3. 点击「**开通**」，选择免费版
4. 创建云环境（名称随意，如：guardian-prod）
5. 等待初始化完成

### 步骤2：创建数据库集合（2分钟）

在云开发控制台：

1. 点击「**数据库**」标签
2. 点击「**+**」号创建集合
3. 输入集合名称，点击确定

**需要创建的11个集合：**

```
applications
users
interviews
training_videos
training_records
practice_questions
exam_questions
exam_records
notices
sms_codes
activities
```

**设置权限（测试环境）：**
- 每个集合 → 权限设置 → 选择「所有用户可读写」

### 步骤3：部署云函数（2分钟）

在开发者工具左侧文件树：

1. 展开 `cloudfunctions` 文件夹
2. 右键点击每个云函数文件夹
3. 选择「**上传并部署：云端安装依赖**」

**需要部署的云函数：**

```
✅ login                  (登录认证)
✅ sendSMS                (发送验证码)
✅ updateVideoProgress    (更新视频进度)
✅ getExamQuestions       (获取考试题目)
✅ submitExam             (提交考试)
✅ checkGuardian          (查询监护人)
✅ getStatistics          (获取统计数据)
✅ seedTestData          (添加测试数据 - 可选)
```
